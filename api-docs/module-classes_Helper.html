    static reverse(string) → {string}
    classes/Helper.js, line 69

    Reversing strings containing especial unicode characters can cause problems using usual ways to reverse! For example this string: 'foo bar maana manana' will be corrupt if used string.split("").reverse().join(""); We'll use following code from mathiasbynens to reverse a string properly repo source

    Parameters:
    Name Type Description
    string

    An string to reverse
    Returns:
    string
    Example
    let str = 'foo  bar maana manana';
 let correctlyReversed = reverse(str); // 'ananam anaam rab  oof'
 let corruptedReverse  = str.split("").reverse().join(""); // 'ananam anaam rab  oof'
